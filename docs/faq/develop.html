Developer Edition FAQ

Introduction

This is an FAQ regarding problems you may encounter while developing with KiriKiri Z.


Distribution
Q. I want to change the icon of the exe.
A. KiriKiri Z does not include a standard icon-changing tool.
Please change the icon in Visual Studio and rebuild, or use free software that can modify resources to change the icon.
Please search for terms such as "free software to rewrite exe resources".

Movies
Q. When using MPEG 1, the aspect ratio becomes incorrect if the width exceeds 1024.
A. In Mixer (VMR) or Overlay mode, there appears to be a bug in the tjsmovie implementation where the aspect ratio may become incorrect depending on the environment.
Please use Layer mode, or use WMV or H.264.
Q. When playing a WMV in Overlay mode, it is inverted vertically.
A. This appears to happen depending on the environment.
Please ensure you use Mixer (VMR) mode.

Save Data
Q. Even after modifying the script, the changes are not reflected.
A. It is possible that old save data remains in the savedata folder, and behavior is being affected by that data.
Deleting the entire savedata folder and then re-running may resolve this.

Various Tools
Q. Can I use the included tools for purposes other than data used in KiriKiri Z?
A. There are no particular restrictions on the usage of the included tools.
