User Edition FAQ

Introduction

This is an FAQ regarding problems that users playing the game may encounter.


Startup
Q. An error message says MFPlat.DLL or MF.dll cannot be found.
A. In the case of Windows N editions, the game cannot start because multimedia-related features are not included.
Please install the Media Feature Pack.

Sound
Q. When a voice is played, there is a "pop" sound at the beginning.
A. This may be fixed with -wsrecreate=yes.
Specify it directly in the command line options, or set "Sound: DirectSound Secondary Buffer Recreation: Always Recreate" in the Kirikiri Configuration.

Mouse / Input
Q. The message does not advance or respond even when turning the wheel.
A. This may be fixed with -wheel.
Try both "dinput" and "message" and choose the one that works better.
Specify it directly in the command line options, or set "Input: Mouse Wheel Detection Method: DirectInput" or "Input: Mouse Wheel Detection Method: Window Message" in the Kirikiri Configuration.
Q. Messages advance on their own even though I'm not doing anything.
A. If a gamepad is connected and its button is held down, the messages may advance due to key repeat.
Also, they will advance automatically if the game is set to auto-read mode.
