KWidget.onSizeModified

Function/Meaning
Hook function called when the widget's own size is changed due to an update of the widget's minimum or maximum size.
Type
Method of the KWidget class
Syntax
onSizeModified(newWidth, newHeight, oldWidth, oldHeight)
Arguments
newWidth New width
newHeight New height
oldWidth Width before change
@param oldHeight Height before change
Return Value
None (void)
Description
