KVirtualWidget.onDraw

Function/Meaning
Draw request. This callback function is called when a screen redraw is required due to reasons such as changes in the size of the scroll area or screen scrolling. Implement this in a derived class to redraw the elements within the specified rectangle. Since the rectangle is expressed in the virtual coordinate system, it is recommended to perform the drawing process using virtual coordinate system drawing functions that have the "virtual" prefix. At the time this function is called, the rectangular area is clipped, so it is fine to issue drawing commands that extend outside the area.
Type
Method of the KVirtualWidget class
Syntax
onDraw(vleft, vtop, vwidth, vheight)
Arguments
vleft Left coordinate (virtual coordinate system)
vtop Top coordinate (virtual coordinate system)
vwidth Width (virtual coordinate system)
vheight Height (virtual coordinate system)
Return Value
None (void)
