KListItem.onValueModified

Function/Meaning
Hook function called when the "value" changes. In widgets that have a "value", it is called when the value is modified. By default, it is set to call the parent's onChildValueModified.
Type
Method of KListItem class
Syntax
onValueModified(newValue, oldValue)
Arguments
newValue New value
oldValue Value before change
Return value
None (void)
Description
