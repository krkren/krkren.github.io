KWidget.onViewModified

Function/Meaning
Callback called when the view rectangle is updated
Type
Method of the KWidget class
Syntax
onViewModified(vleft, vtop, vwidth, vheight)
Arguments
vleft Left edge of the rectangle
vtop Top edge of the rectangle
vwidth Rectangle width
vheight Rectangle height

A callback function called when the view rectangle is updated.
It is not always called; it is only called when registered
as a child of a specific Widget.

The list of Widgets that trigger this callback is as follows:
- KScrollLayout
- KRelativeLayout
Return Value
None (void)
