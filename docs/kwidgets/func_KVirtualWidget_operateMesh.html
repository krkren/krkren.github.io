```html operateMesh - Mesh Composition

KVirtualWidget.operateMesh

Function/Meaning
Mesh Composition
Type
Method of the KVirtualWidget class
Syntax
operateMesh(src, sleft, stop, swidth, sheight, mesh, divisionX, divisionY, mode, opa, type)
Arguments
src Specifies the source layer object.
sleft Specifies the left position of the rectangle to copy (relative to the image position of the source layer) in pixels.
stop Specifies the top position of the rectangle to copy (relative to the image position of the source layer) in pixels.
swidth Specifies the width of the rectangle to copy (relative to the image position of the source layer) in pixels.
sheight Specifies the height of the rectangle to copy (relative to the image position of the source layer) in pixels.
mesh Mesh.
divisionX Number of divisions in the X direction. Specifies how many polygons to divide the X direction into. 1 = No division.
divisionY Number of divisions in the Y direction. Specifies how many polygons to divide the Y direction into. 1 = No division.
mode Specifies the operation mode.
opa Specifies the opacity for blending (0 to 255).
type Specifies the type of affine transformation.
Return Value
None (void)
Description
```
