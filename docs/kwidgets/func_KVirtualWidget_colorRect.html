KVirtualWidget.colorRect

Function/Meaning
Rectangle overlay
Type
Method of the KVirtualWidget class
Syntax
colorRect(left, top, width, height, value, opa)
Arguments
left Specifies the left edge position of the rectangle to fill in pixels (relative to image position).
top Specifies the top edge position of the rectangle to fill in pixels (relative to image position).
width Specifies the width of the rectangle to fill in pixels (relative to image position).
height Specifies the height of the rectangle to fill in pixels (relative to image position).
value Specifies the color or value to fill with.
opa Specifies the fill opacity (-255 to 0 to 255).
Return Value
None (void)
Description
