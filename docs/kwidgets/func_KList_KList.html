KList.KList

Function/Meaning
List View. A list view that supports nesting. It consists of the list body KList and list items KListItem. KList is a class that inherits from KWidget, but KListItem is not a KWidget. From the perspective of memory conservation, KListItem is designed to internally create a proxy widget only when it is displayed on screen. However, because most mouse-related and drag-and-drop events are forwarded from the proxy widget to KListItem, there are effectively no issues with treating KListItem as a KWidget-inherited class.
