KRelativeLayout.KRelativeLayout

Function/Meaning
Absolute position layout. Places widgets within the layout based on relative coordinates. This enables so-called "multi-layer scrolling." By registering a KRelativeLayout within a KScrollLayout and placing multiple widgets of different sizes inside, the scroll amount is automatically adjusted according to their relative positions.
Type
KRelativeLayout class method
Syntax
KRelativeLayout(win, name)
Arguments
win Window
name Name
Return Value
None (void)
