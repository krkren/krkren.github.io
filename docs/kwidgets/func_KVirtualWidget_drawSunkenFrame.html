KVirtualWidget.drawSunkenFrame

Function/Meaning
Draws a sunken frame
Type
Method of the KVirtualWidget class
Syntax
drawSunkenFrame(left, top, width, height, boxColor)
Arguments
left Left edge of the frame
top Top edge of the frame
width Width of the frame
height Height of the frame
boxColor Background color to fill the frame
Return Value
None (void)
Description
