KTextEditRow.pointerPos

Function/Meaning
Returns the pointer position corresponding to the character index in the parent coordinate system
Type
Method of the KTextEditRow class
Syntax
pointerPos(index, tail)
Arguments
index Index
tail Whether it is the end of the line (For lines that wrap automatically at the right edge, the right edge of the wrap and the left edge of the next line represent the same pointer position. This option selects which coordinate to obtain.)
Return Value
None (void)
Description
