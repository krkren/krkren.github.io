KWindow.onChildMarginModified

Function/Meaning
Hook function called when the primary widget's margin is modified
Type
Method of the KWindow class
Syntax
onChildMarginModified(child, newMarginLeft, newMarginTop, newMarginRight, newMarginBottom, oldMarginLeft, oldMarginTop, oldMarginRight, oldMarginBottom)
Arguments
child The child widget whose margin was modified
newMarginLeft New left margin
newMarginTop New top margin
newMarginRight New right margin
newMarginBottom New bottom margin
oldMarginLeft Left margin before modification
oldMarginTop Top margin before modification
oldMarginRight Right margin before modification
oldMarginBottom Bottom margin before modification
Return Value
None (void)
Description
