KVirtualWidget.drawText

Function/Meaning
Draw text
Type
Method of the KVirtualWidget class
Syntax
drawText(x, y, text, color, opa, aa, shadowlevel, shadowcolor, shadowwidth, shadowofsx, shadowofsy)
Arguments
x Specifies the x-coordinate of the origin (at the image position) where text drawing starts, in pixels, in the virtual coordinate system.
y Specifies the y-coordinate of the origin (at the image position) where text drawing starts, in pixels, in the virtual coordinate system.
text Specifies the text to be drawn.
color Specifies the color of the text to be drawn in 0xRRGGBB format.
opa Specifies the opacity of the text to be drawn (-255 to 0 to 255).
aa Specifies whether to perform anti-aliasing.
shadowlevel Specifies the opacity of the shadow. The appropriate value varies depending on the value of the shadowwidth argument.
shadowcolor Specifies the color of the shadow in 0xRRGGBB format.
shadowwidth Specifies the width (blur) of the shadow. 0 is the sharpest (no blur), and larger values blur the shadow.
shadowofsx Specifies the x-coordinate value of the shadow position in pixels. If 0 is specified, the shadow is drawn directly behind.
shadowofsy Specifies the y-coordinate value of the shadow position in pixels. If 0 is specified, the shadow is drawn directly behind.
Return Value
None (void)
Description
