KWindow.onChildAlignModified

Function/Meaning
Hook function called when the alignment of a primary widget is modified
Type
Method of the KWindow class
Syntax
onChildAlignModified(child, newAlignHorizontal, newAlignVertical, oldAlignHorizontal, oldAlignVertical)
Arguments
child The child widget whose alignment has changed
newAlignHorizontal New horizontal alignment
newAlignVertical New vertical alignment
oldAlignHorizontal Previous horizontal alignment
oldAlignVertical Previous vertical alignment
Return Value
None (void)
Description
