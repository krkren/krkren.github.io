KContextMenu.show

Function/Meaning
Display Menu. When called with a base layer and base coordinates specified, the context menu opens at that position. If omitted, it opens at the current cursor position.
Type
Method of the KContextMenu class
Syntax
show(baseLayer, x, y)
Arguments
baseLayer Base display layer
x Base display X-coordinate
y Base display Y-coordinate
Return Value
None (void)
