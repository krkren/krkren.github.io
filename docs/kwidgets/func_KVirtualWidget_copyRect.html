KVirtualWidget.copyRect

Function/Meaning
Rectangle Copy
Type
KVirtualWidget class method
Syntax
copyRect(left, top, src, sleft, stop, swidth, sheight)
Arguments
left Specifies the left position of the destination rectangle in pixels (relative to the destination layer's image position) in the virtual coordinate system.
top Specifies the top position of the destination rectangle in pixels (relative to the destination layer's image position) in the virtual coordinate system.
src Specifies the source layer object.
sleft Specifies the left position of the rectangle to be copied in pixels (relative to the source layer's image position).
stop Specifies the top position of the rectangle to be copied in pixels (relative to the source layer's image position).
swidth Specifies the width of the rectangle to be copied in pixels (relative to the source layer's image position).
sheight Specifies the height of the rectangle to be copied in pixels (relative to the source layer's image position).
Return Value
None (void)
Description
