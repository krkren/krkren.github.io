KListItem.onChildValueModified

Function/Meaning
Hook function called when a child widget's "value" changes. This is called when the "value" of a child widget is modified. By default, it is set to further call the parent's onChildValueModified. Unless a widget somewhere overrides the function and performs processing, the notification of the value change will reach KWindow.
Type
Method of KListItem class
Syntax
onChildValueModified(child, newValue, oldValue)
Arguments
newValue New value
oldValue Value before change
Return Value
None (void)
