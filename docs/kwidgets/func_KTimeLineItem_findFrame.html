KTimeLineItem.findFrame

Function/Meaning
Search for a frame corresponding to the time
Type
KTimeLineItem class method
Syntax
findFrame(time, includeTail)
Arguments
time Time
includeTail Whether to include the trailing space after the last frame in the search
Return Value
Index\n\nReturns the index of the frame corresponding to the time.\nReturns -1 if the frame was not found.
