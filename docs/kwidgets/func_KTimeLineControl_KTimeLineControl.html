KTimeLineControl.KTimeLineControl

Function/Meaning
Constructor
Type
Method of the KTimeLineControl class
Syntax
KTimeLineControl(window, name)
Arguments
window Window
name Name

Creates a timeline control.

Objects of KTimeLineItem derived classes can be registered using the following functions.
Since KTimeLineItem is a derived class of KListItem, it can form nested lists just like regular list items.

- childCount: Counts child items
- childAt(index): References a child item
- add(item): Adds a child item
- insert(index, item): Inserts a child item
- remove(item): Removes a child item
- removeAll(doInvalidate): Removes all child items

The following functions can be used to control the behavior of the control:

- label: Label
- resize: Whether to display resize handles

Timeline parameters can be retrieved or set with the following functions:

- selectedTime: The currently pointed time
- selectedLayer: The currently pointed layer
- loopBeginTime: Sets or gets the loop start time
- loopEndTime: Sets or gets the loop end time
- setLoopTime(beginTime, endTime): Sets the loop start and end points simultaneously

The timeline control has a timeline playback function.
Control it using the following functions:

- framePerSecond: Sets the fps
- playing: Whether playback is in progress.
- play(): Starts playback
- stop(): Stops playback

The following event functions are called at specific timings:

- onTimeLocatorRightClick(time): Called when the time locator is right-clicked
- onTimeLineSelectionRightClick(): Called when the timeline is right-clicked while in a selected state
- onTimeLineSelectionKeyDown(key, shift): Called when a key is pressed while the timeline is in a selected state
- onMultipleFrameDragged(): Multiple frames were dragged together
- onRenderFrame(): Called when a frame needs to be rendered
- onTimeSelect(time): Called when the pointed time changes
- onLayerSelect(layer): Called when the pointed layer changes
- onPlay(): Called when playback starts
- onStop(playOverLastFrame): Called when playback stops
Return Value
None (void)
