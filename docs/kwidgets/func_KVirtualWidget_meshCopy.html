KVirtualWidget.meshCopy

Function/Meaning
Mesh copy
Type
Method of the KVirtualWidget class
Syntax
meshCopy(src, sleft, stop, swidth, sheight, mesh, divisionX, divisionY, type, clear)
Arguments
src Specifies the source layer object.
sleft Specifies the left edge position of the rectangle to be copied (within the image coordinates of the source layer) in pixels.
stop Specifies the top edge position of the rectangle to be copied (within the image coordinates of the source layer) in pixels.
swidth Specifies the width of the rectangle to be copied (within the image coordinates of the source layer) in pixels.
sheight Specifies the height of the rectangle to be copied (within the image coordinates of the source layer) in pixels.
mesh Mesh.
divisionX Number of divisions in the X direction. Specifies how many polygons to divide the X direction into. 1 = No division.
divisionY Number of divisions in the Y direction. Specifies how many polygons to divide the Y direction into. 1 = No division.
type Specifies the type of affine transformation.
clear Specifies whether to clear the surrounding area of the affine-transformed image in the destination layer with the color and transparency indicated by the Layer.neutralColor property.
Return Value
None (void)
Description
