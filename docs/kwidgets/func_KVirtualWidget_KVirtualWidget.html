KVirtualWidget.KVirtualWidget

Function/Meaning
Constructor. Creates a virtual widget. A virtual widget is a virtual layer object that holds only the area actually displayed on the screen during scrolling as its canvas. It allows for a very large canvas area to be secured without worrying about the size of the layer. The virtual widget itself is declared as a layer holder with hasImage = false. Since images are drawn on the canvas, mouse events are set to "always receive" regardless of the alpha value of the images on the canvas.
