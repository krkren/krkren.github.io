KVirtualWidget.operateRect

Function/Meaning
Rectangular operation composition
Type
Method of KVirtualWidget class
Syntax
operateRect(left, top, src, sleft, stop, swidth, sheight, mode, opa)
Arguments
vleft left: Specifies the left edge position of the destination rectangle in pixels (relative to the destination layer's image position) in the virtual coordinate system.
vleft top: Specifies the top edge position of the destination rectangle in pixels (relative to the destination layer's image position) in the virtual coordinate system.
vleft src: Specifies the source layer object.
vleft sleft: Specifies the left edge position of the source rectangle in pixels (relative to the source layer's image position).
vleft stop: Specifies the top edge position of the source rectangle in pixels (relative to the source layer's image position).
vleft swidth: Specifies the width of the source rectangle in pixels (relative to the source layer's image position).
vleft sheight: Specifies the height of the source rectangle in pixels (relative to the source layer's image position).
vleft mode: Specifies the operation mode.
vleft opa: Specifies the intensity of the operation (0 to 255).
Return Value
None (void)
Description
