KVirtualWidget.piledCopy

Function/Meaning
Copy an image with layers superimposed
Type
Method of the KVirtualWidget class
Syntax
piledCopy(left, top, src, sleft, stop, swidth, sheight)
Arguments
left Specifies the left edge position of the destination rectangle in pixels (relative to the image position of the destination layer) in the virtual coordinate system.
top Specifies the top edge position of the destination rectangle in pixels (relative to the image position of the destination layer) in the virtual coordinate system.
src Specifies the source layer object.
sleft Specifies the left edge position of the rectangle to be copied in pixels (relative to the display position of the source layer).
stop Specifies the top edge position of the rectangle to be copied in pixels (relative to the display position of the source layer).
swidth Specifies the width of the rectangle to be copied in pixels (relative to the display position of the source layer).
sheight Specifies the height of the rectangle to be copied in pixels (relative to the display position of the source layer).
Return Value
None (void)
Description
