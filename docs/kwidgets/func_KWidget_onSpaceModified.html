KWidget.onSpaceModified

Function/Meaning
Hook function called when spacing is modified
Type
Method of the KWidget class
Syntax
onSpaceModified(newSpaceHorizontal, newSpaceVertical, oldSpaceHorizontal, oldSpaceVertical)
Arguments
newSpaceHorizontal New horizontal spacing
newSpaceVertical New vertical spacing
oldSpaceHorizontal Horizontal spacing before modification
newSpaceVertical Vertical spacing before modification
Return Value
None (void)
Description
