KVirtualWidget.drawImage

Function/Meaning
Draws an image. The destination position accounts for the source image's Bounds, and the size is specified in pixels.
Type
Method of KVirtualWidget class
Syntax
drawImage(dleft, dtop, src)
Arguments
x Destination X origin
y Destination Y origin
image Source image (Image/Layer/Filename)
Return Value
Update region information (RectF)
