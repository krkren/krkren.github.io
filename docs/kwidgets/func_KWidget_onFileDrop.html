KWidget.onFileDrop

Function/Meaning
File dropped from outside. The default behavior is to call the parent's onFileDrop. The message is chained sequentially from child to parent until some widget overrides the function and performs processing.
Type
Method of the KWidget class
Syntax
onFileDrop(x, y, files, shift)
Arguments
x The mouse X-coordinate when the file was dropped
y The mouse Y-coordinate when the file was dropped
files An Array object containing the names of the dropped files.
shift The state of the shift-type modifier keys held down at the same time.
Return Value
None (void)
