KWidget.onChildMinMaxSizeModified

Function/Meaning
Hook function called when a child widget's minimum or maximum size is updated
Type
KWidget class method
Syntax
onChildMinMaxSizeModified(child, newMinWidth, newMinHeight, newMaxWidth, newMaxHeight, oldMinWidth, oldMinHeight, oldMaxWidth, oldMaxHeight)
Arguments
child The child widget whose minimum/maximum size was updated
newMinWidth New minimum width
newMinHeight New minimum height
newMaxWidth New maximum width
oldMinWidth Minimum width before change
oldMinHeight Minimum height before change
oldMaxWidth Maximum width before change
oldMaxHeight Maximum height before change
Return Value
None (void)
Explanation
