KWindow.onChildValueModified

Function/Meaning
Hook function called when a widget's "value" changes. It is called when the "value" of a widget is modified. If you override the widget's own onValueChanged, or override onChildValueModified in any of the widget's ancestors, the notification will not propagate to the window.
Type
Method of the KWindow class
Syntax
onChildValueModified(child, newValue, oldValue)
Arguments
child The child widget whose value was changed
newValue The new value
oldValue The value before the change
Return Value
None (void)
