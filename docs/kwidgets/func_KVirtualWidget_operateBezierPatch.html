KVirtualWidget.operateBezierPatch

Function/Meaning
Bezier patch composition
Type
Method of KVirtualWidget class
Syntax
operateBezierPatch(src, sleft, stop, swidth, sheight, bezierPatch, divisionX, divisionY, mode, opa, type)
Arguments
src Specifies the source layer object.
sleft Specifies the left position of the rectangle to copy in pixels (relative to the image position of the source layer).
stop Specifies the top position of the rectangle to copy in pixels (relative to the image position of the source layer).
swidth Specifies the width of the rectangle to copy in pixels (relative to the image position of the source layer).
sheight Specifies the height of the rectangle to copy in pixels (relative to the image position of the source layer).
bezierPatch Bezier patch. Specifies the coordinates of 16 control points as an array of 32 decimal values.
divisionX Number of divisions in the X direction. Specifies how many polygons to divide the X direction into. 1 = no division.
divisionY Number of divisions in the Y direction. Specifies how many polygons to divide the Y direction into. 1 = no division.
mode Specifies the operation mode.
opa Specifies the opacity for blending (0 to 255).
type Specifies the type of affine transformation.
Return Value
None (void)
Description
