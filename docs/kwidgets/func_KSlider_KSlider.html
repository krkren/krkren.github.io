KSlider.KSlider

Function/Meaning
Constructor. Creates a slider. If the constant SLIDER_ADAPTIVE_WIDTH is specified for w, the slider automatically calculates and sets a minimum width sufficient to allow all values from the minimum to the maximum to be specified by sliding.
Type
Method of the KSlider class
Syntax
KSlider(win, w, h, leftValue, rightValue, step, name)
Arguments
win Window
w Slider width
h Slider height
minValue Minimum value
maxValue Maximum value
name Name
step Step interval
Return Value
None (void)
Description
