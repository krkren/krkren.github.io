KMenuItem.missing

Function/Meaning
Searches for a child widget with a specific name from the widget tree. Instead of going through the find function, you can refer to a widget in the widget tree by specifying the name as an object property. Note that for performance reasons, if a symbol with the same name exists globally, it will prioritize looking at that.
Type
Method of the KMenuItem class
Syntax
missing(get_or_set, name, value)
Arguments
Return Value
None (void)
