KVirtualWidget.drawString

Function/Meaning
Draws a string. Rendering quality follows textRenderingHint.
Type
KVirtualWidget class method
Syntax
drawString(font, app, x, y, text)
Arguments
font Font. Specifies the font with GdiPlus.Font
app Appearance. Specifies the drawing style with GdiPlus.Appearance. *Only the brush is referenced.
x Drawing position X
y Drawing position Y
text Text to draw
Return Value
Update region information (RectF)
Description
