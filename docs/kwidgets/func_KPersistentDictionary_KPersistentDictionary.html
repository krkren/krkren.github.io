KPersistentDictionary.KPersistentDictionary

Function/Meaning
Constructor. Initializes a persistent dictionary. If values were saved in the dictionary during a previous execution, those values are read and used as the initial values.
Type
KPersistentDictionary class method
Syntax
KPersistentDictionary(filePath, virtual)
Arguments
filePath File path to save the dictionary
Return Value
None (void)
