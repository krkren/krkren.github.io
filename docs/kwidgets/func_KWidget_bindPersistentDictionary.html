KWidget.bindPersistentDictionary

Function/Meaning
Binds a persistent dictionary. Binds the widget to an object of the KPersistentDictionary class. By default, it searches all child widgets and recursively calls bindPersistentDictionary for those widgets.
Type
Method of KWidget class
Syntax
bindPersistentDictionary(dict)
Arguments
dict Dictionary to bind
Return Value
None (void)
