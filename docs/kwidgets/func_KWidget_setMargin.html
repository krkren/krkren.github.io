KWidget.setMargin

Function/Meaning
Sets margins. Sets margins all at once. If only one argument is specified, all margins (top, bottom, left, and right) are set to the same value. If only two arguments are specified, the first argument sets the left/right margins, and the second sets the top/bottom margins. If all four arguments are specified, each margin is set individually.
Type
Method of the KWidget class
Syntax
setMargin(l, t, r, b)
Arguments
l Left margin
t Top margin
r Right margin
b Bottom margin
Return Value
None (void)
