KWidget.drawChecker

Function/Meaning
Draws a checkerboard pattern
Type
Method of the KWidget class
Syntax
drawChecker(l, t, w, h, csize, color1, color2, sx, sy)
Arguments
l Left edge of the checkerboard
t Top edge of the checkerboard
w Width of the checkerboard
h Height of the checkerboard
csize Checker size
color1 Color 1
color2 Color 2
sx Shift X
sy Shift Y
Return Value
None (void)
Description
