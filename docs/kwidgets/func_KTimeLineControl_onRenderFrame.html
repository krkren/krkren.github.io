KTimeLineControl.onRenderFrame

Function/Meaning
Frame rendering required. Called when it is determined that frame rendering is necessary. Specifically, it applies to the following situations: - When a new frame is added to an item - When a frame is deleted from an item - When a keyframe is inserted into an item - When a keyframe is deleted from an item - When an item is unserialized - When an item is dragged and moved - When the pointed time is changed - When the pointed layer is changed
Type
Method of KTimeLineControl class
Syntax
onRenderFrame()
Arguments
Return Value
None (void)
Description
