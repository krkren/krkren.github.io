KListItem.missing

Function/Meaning
Searches for a child widget with a specific name from descendant items. You can reference widgets within descendant items by specifying the name as an object property without going through the find function. Note that for performance reasons, if a symbol with the same name exists globally, it will prioritize looking there.
Type
Method of the KListItem class
Syntax
missing(get_or_set, name, value)
Arguments
Return Value
None (void)
