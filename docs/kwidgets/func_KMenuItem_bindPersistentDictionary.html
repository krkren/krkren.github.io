KMenuItem.bindPersistentDictionary

Function/Meaning
Binds a persistent dictionary. Binds a widget to an object of the KPersistentDictionary class. By default, it searches all child widgets and recursively calls bindPersistentDictionary on those widgets.
Type
KMenuItem class method
Syntax
bindPersistentDictionary(dict)
Arguments
dict The dictionary to bind
Return Value
None (void)
