KListItem.isAcceptRename

Function/Meaning
Determines whether to allow renaming a label. A user-defined function that determines whether a specified name is acceptable during a rename operation using renameAsUserInput. The default implementation always returns true.
Type
Method of the KListItem class
Syntax
isAcceptRename(label)
Arguments
label The label to evaluate
Return Value
Returns true if accepted, or false if rejected.
