KWidget.getWidgetAt

Function/Meaning
Get the widget at the specified position
Type
Method of KWidget class
Syntax
getWidgetAt(x, y, exclude_self, get_disabled)
Arguments
x Specifies the x-coordinate of the widget position you want to get, in pixels, on the display coordinates.
y Specifies the y-coordinate of the widget position you want to get, in pixels, on the display coordinates.
exclude_self Specifies whether to exclude itself from the widget search.
get_disabled Specifies whether to retrieve objects of disabled widgets.
Return Value
None (void)
Description
