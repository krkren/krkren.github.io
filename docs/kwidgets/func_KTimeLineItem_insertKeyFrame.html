KTimeLineItem.insertKeyFrame

Function/Meaning
Inserting a frame
Type
Method of the KTimeLineItem class
Syntax
insertKeyFrame(time, type, content)
Arguments
time Time
type Type
content Initial content.

Inserts a new frame.

If there is an existing frame at the specified location, it will be overwritten.
In the case of an empty frame, the content must be explicitly specified.
Return Value
None (void)
Description
