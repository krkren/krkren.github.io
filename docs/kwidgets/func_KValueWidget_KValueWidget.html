KValueWidget.KValueWidget

Function/Meaning
Constructor. A generic term for widgets that have a "value" property. By calling bindPersistentDictionary() and binding it to a property, the value can be persisted.
Type
Method of the KValueWidget class
Syntax
KValueWidget(win, name)
Arguments
win Window
name Name. Persistence will not be performed unless a unique name is specified.
Return Value
None (void)
Description
