KVirtualWidget.affineCopy

Function/Meaning
Affine Copy
Type
Method of KVirtualWidget class
Syntax
affineCopy(src, sleft, stop, swidth, sheight, affine, A, B, C, D, E, F, type, clear)
Arguments
src Specifies the source layer object.
sleft Specifies the left position of the rectangle to be copied (in the image coordinates of the source layer) in pixels.
stop Specifies the top position of the rectangle to be copied (in the image coordinates of the source layer) in pixels.
swidth Specifies the width of the rectangle to be copied (in the image coordinates of the source layer) in pixels.
sheight Specifies the height of the rectangle to be copied (in the image coordinates of the source layer) in pixels.
affine Specifies how the following six arguments (parameters A to F) are handled.
A Parameter A. Interpretation changes depending on the affine argument.
B Parameter B. Interpretation changes depending on the affine argument.
C Parameter C. Interpretation changes depending on the affine argument.
D Parameter D. Interpretation changes depending on the affine argument.
E Parameter E. Interpretation changes depending on the affine argument.
F Parameter F. Interpretation changes depending on the affine argument.
type Specifies the type of affine transformation.
clear Specifies whether to clear the area surrounding the affine-transformed image in the destination layer with the color/opacity specified by the Layer.neutralColor property.
Return Value
None (void)
Description
