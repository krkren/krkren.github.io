KVirtualWidget.redrawRect

Function/Meaning
Forced redraw. Issues a drawing process for the specified area forcibly, separately from drawing requests that occur automatically due to scrolling.
Type
Method of KVirtualWidget class
Syntax
redrawRect(vleft, vtop, vwidth, vheight)
Arguments
vleft Left coordinate (virtual coordinate system)
vtop Top coordinate (virtual coordinate system)
vwidth Width (virtual coordinate system)
vheight Height (virtual coordinate system)
Return Value
None (void)
