.queryColorPicker

Function/Meaning
Function to call the color picker. Displays a color picker dialog to edit a color. When hasAlpha=false, it handles color values in 0xRRGGBB format; when hasAlpha=true, it handles color values in 0xAARRGGBB format. If the dialog is cancelled, it returns void instead of a color.
Type
Global method
Syntax
queryColorPicker(color, hasAlpha, dict, name)
Arguments
color Color
hasAlpha Whether it has alpha?
dict Dictionary to save the color picker position
name Name of the color picker
Return Value
The modified color
