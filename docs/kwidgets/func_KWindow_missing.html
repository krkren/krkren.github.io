KWindow.missing

Function/Meaning
Searches for a child widget with a specific name from the widget tree. You can reference widgets in the widget tree by specifying the name as an object property without going through the find function. Note that for performance reasons, if a global symbol with the same name exists, it will prioritize looking that up.
Type
Method of the KWindow class
Syntax
missing(get_or_set, name, value)
Arguments
Return Value
None (void)
