KTimeLineItem.getCompleteFrameContent

Function/Meaning
Calculates frame information corresponding to time
Type
Method of the KTimeLineItem class
Syntax
getCompleteFrameContent(time)
Arguments
time time
Return Value
Frame information\n\nCalculates and returns frame information corresponding to the time.\nIf there is an interpolatable tween frame, it returns the frame information with the tween completed.\nIf there is no interpolatable frame, it returns the keyframe's frame information as is.\nReturns void if there is no frame at the specified point.
