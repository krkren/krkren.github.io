KFixedSizeLayout.KFixedSizeLayout

Function/Meaning
Constructor. A layout that locks the "maximum size" of children in the vertical or horizontal direction to always remain the "minimum size".
Type
Method of the KFixedSizeLayout class
Syntax
KFixedSizeLayout(window, fixHorizontal, fixVertical, name)
Arguments
window Window
fixHorizontal Horizontal lock
fixVertical Vertical lock
name Name
Return Value
None (void)
