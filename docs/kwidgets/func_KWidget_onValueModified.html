KWidget.onValueModified

Function/Meaning
Hook function called when the "value" changes. This is called in widgets that have a "value" when that value is modified. By default, it is set to call the parent's onChildValueModified.
Type
Method of the KWidget class
Syntax
onValueModified(newValue, oldValue)
Arguments
newValue New value
oldValue Value before the change
Return Value
None (void)
Description
