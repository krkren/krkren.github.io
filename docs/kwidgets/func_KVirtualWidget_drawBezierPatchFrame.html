KVirtualWidget.drawBezierPatchFrame

Function/Meaning
Drawing a Bezier patch frame
Type
Method of the KVirtualWidget class
Syntax
drawBezierPatchFrame(outerApp, innerApp, bezierPatch)
Arguments
outerApp The appearance for drawing the outer frame. Specify the drawing style with GdiPlus.Appearance.
innerApp The appearance for drawing the inner frame. Specify the drawing style with GdiPlus.Appearance.
bezierPatch Bezier patch. Specify the coordinate values of 16 control points as an array of 32 floating-point values.
Return Value
None (void)
Description
