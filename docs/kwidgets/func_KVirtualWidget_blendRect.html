KVirtualWidget.blendRect

Function/Meaning
Rectangular Blend
Type
Method of KVirtualWidget class
Syntax
blendRect(left, top, src, sleft, stop, swidth, sheight, opa)
Arguments
left Specifies the left edge position of the destination rectangle in pixels (relative to the image position of the destination layer) in the virtual coordinate system.
top Specifies the top edge position of the destination rectangle in pixels (relative to the image position of the destination layer) in the virtual coordinate system.
src Specifies the source layer object for blending.
sleft Specifies the left edge position of the rectangle to be blended in pixels (relative to the image position of the source layer).
stop Specifies the top edge position of the rectangle to be blended in pixels (relative to the image position of the source layer).
swidth Specifies the width of the rectangle to be blended in pixels (relative to the image position of the source layer).
sheight Specifies the height of the rectangle to be blended in pixels (relative to the image position of the source layer).
opa Specifies the opacity of the blend (0 to 255).
Return Value
None (void)
Description
