File Format

TypeFormatExtensionDescription
Still Image Windows Bitmap .bmp Only uncompressed formats are supported.
In the case of 32-bit bitmaps, they are treated as having an alpha channel.
JPEG .jpg
.jpeg		 Decoding precision can be specified.
Settable values are 'high', 'normal', or 'low'.
The default is 'normal'.
'low' uses fixed-point (integer) AAN for IDCT, with no chroma upsampling interpolation.
'normal' uses fixed-point (integer) LLM for IDCT, with chroma upsampling interpolation.
'high' uses floating-point AAN for IDCT, with chroma upsampling interpolation.
For details on how to specify, please refer to Startup Option -jpegdec (JPEG image decoding precision).
Portable Network Graphic (PNG) .png Supports alpha channels and transparent color specifications.
TLG5 .tlg Kirikiri original format.
Characterized by high expansion speed (about 4 times faster than PNG).
Can be converted with the Image Format Converter.
TLG6 .tlg Kirikiri original format.
Higher compression ratio than PNG or TLG5, but takes about twice as long to expand as TLG5.
However, it is still about twice as fast as PNG, so it is fast.
Can be converted with the Image Format Converter.
Others .* Can be extended via Susie-plugins.
Audio WAVE .wav Supports uncompressed WAVE format.
Ogg Vorbis .ogg Requires wuvorbis.dll plugin
Others .* Can be extended via plugins.
