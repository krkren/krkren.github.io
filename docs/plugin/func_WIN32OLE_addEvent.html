WIN32OLE.addEvent

Function/Meaning
Registration of event receiver object
Type
Method of the WIN32OLE class
Syntax
addEvent(eventName, receiver)
Arguments
eventName Event type name (e.g., "DWebBrowser2Events2")
receiver Object that receives the events.
A method with the same name as the event held by the object is called.
If there is no corresponding method, no processing is performed.
Return Value
None (void)
Description
