Window.onResizing

Function/Meaning
■Extended Event: Resizing
Type
Window class method
Syntax
onResizing(rect)
Arguments
rect %[ x, y, w, h, type ] Current size being changed (window coordinates)
rect.type Which part of the window is being grabbed for resizing (1 to 8)
Combination of +1: Left, +2: Right, +3: Top, +6: Bottom (Ternary representation)
4 3 5
Layout: 1 3
7 6 8
Return Value
Returning true reflects the changes to rect (forces the position and size).\nIn this case, note that if the rewritten content is not properly adjusted according to the grabbed position (rect.type), the window may move unexpectedly during resizing (see forceResizeRect()).
Description
*Note: In Windows, if the setting to "Show window contents while dragging" is enabled (no size frame display), onResize is called immediately.
