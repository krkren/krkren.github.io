MenuItem.popupEx

Function/Meaning
Pop-up display
Type
MenuItem class method
Syntax
popupEx(flags, x, y, hwnd, rect, menulist)
Arguments
flags
x, y Display position (screen coordinates; defaults to the mouse cursor position if omitted)
hwnd Owner window (defaults to the window the instance belongs to if omitted)
rect Exclusion rectangle area where overlap is forbidden (%[ x, y, w, h ]; defaults to no exclusion area if omitted)
menulist Menu list array (does not need to be instances of MenuItem; an array of dictionaries can also be used)
Return Value
The selected object
Description
Unlike MenuItem.popup, it does not need to belong to a window menu, and you can also pass an array of dictionaries for the list.
Note that icons specified by the bmp* properties will not be displayed in pop-ups created by this function.
