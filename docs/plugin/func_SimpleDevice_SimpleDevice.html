SimpleDevice.SimpleDevice

Function/Meaning
Constructor
Type
Method of SimpleDevice class
Syntax
SimpleDevice(window, width, height)
Arguments
window The parent window.
width Back buffer width. If using a render target, it must be a power of 2.
height Back buffer height. If using a render target, it must be a power of 2.
useRender Set to true to use a render target. Render targets support alpha transparency.

The Irrlicht device does not exist at the moment of creation.
It is initialized before the first event processing after creation, at which point the driver, scene manager, and GUI become available for use.
Processing should wait for the onAttach() event before proceeding.
Return Value
None (void)
