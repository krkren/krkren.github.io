Layer.oozeColor

Function/Meaning
Extends the colors of the layer edges into the transparent areas (prevents false colors when downscaling)
Type
Method of the Layer class
Syntax
oozeColor(level, threshold, fillColor)
Arguments
level The number of times to perform the process. The larger the value, the more the extension area increases.
threshold Alpha threshold (1 to 255). Extends color into pixels with alpha lower than this.
fillColor The fill color for areas outside the processing region (0xRRGGBB).
Return Value
None (void)
Description
