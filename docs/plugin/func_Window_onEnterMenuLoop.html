Window.onEnterMenuLoop

Function/Meaning
■ Extended Event: Notification of Start/End of Menu Processing
Type
Method of the Window class
Syntax
onEnterMenuLoop()
Arguments
Return Value
None (void)
Description
An event notifying the start and end of window menu processing (such as when clicking or pressing F10).
Callback for WM_ENTERMENULOOP / WM_EXITMENULOOP notifications.

* Note that this event does not occur for menus when in full-screen mode!
