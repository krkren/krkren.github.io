Window.sendUserMessage

Function/Meaning
Executes a low-level message transmission.
Type
Method of the Window class
Syntax
sendUserMessage(msg, wparam, lparam)
Arguments
msg Message number
wparam WPARAM (processed as an integer)
lparam LPARAM (processed as an integer)
Return Value
None (void)
Description
The specified message is sent to all Kirikiri instances.
