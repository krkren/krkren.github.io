WaveSoundBuffer.sampleValue

Function/Meaning
Obtaining sample values (new method)
Type
Property of the WaveSoundBuffer class
Description
Obtains samples using getVisBuffer(buf, sampleCount, 1, sampleAhead) and retrieves the maximum value of (value/32768)^2. (Returns a real number from 0 to 1).
*Reading this property implicitly sets useVisBuffer=true.
