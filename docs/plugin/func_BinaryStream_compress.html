BinaryStream.compress

Function/Meaning
Compresses and copies data from another storage
Type
BinaryStream class method
Syntax
compress(storage, elm)
Arguments
storage Source storage (throws an exception if it is the same as the currently open storage)
elm Dictionary for specifying behavior (in/out). Contents are the same as copy(), plus the following:
comp_lv: [in] zlib compression level (1–9). If omitted, or if elm itself is not specified, 9 is used.
Return Value
Number of bytes written
Description
Compresses the target storage using zlib/deflate and writes it from the current stream position.
The filter function is applied to the binary data before compression.
