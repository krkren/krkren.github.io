PSD.getBlend

Function/Meaning
Get blending results. The retrieval area must fit within the overall image size.
Type
PSD class method
Syntax
getBlend(layer, left, top, width, height)
Arguments
layer Storage destination layer (will be adjusted to width and height size)
left The left coordinate of the blending result retrieval area
top The top coordinate of the blending result retrieval area
width Retrieval width
height Retrieval height
Return Value
Returns true if the retrieval is successful
Description
