Process

Provides a mechanism to launch external applications and wait for their termination.

Members

Methods
commandExecute (Execution of command line programs with console output capture )
execute (Execution of external programs in the background )
onExecuted (Event: Shell execution finished )
onOutput (Event: Console output )
sendSignal (Sending Ctrl+C event (experimental: may not work in all environments) )
terminate (Forced termination of a running external program )
Properties
status (Current status )
