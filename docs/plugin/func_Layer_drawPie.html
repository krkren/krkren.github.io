Layer.drawPie

Function/Meaning
Draws a pie (sector of an ellipse)
Type
Method of Layer class
Syntax
drawPie(app, x, y, width, height, startAngle, sweepAngle)
Arguments
app Specifies the drawing style with GdiPlus.Appearance
x Left coordinate
y Top coordinate
width Width
height Height
startAngle Clockwise arc starting position (angle)
sweepAngle Sweep angle (angle to draw)
Return Value
Update region information (RectF)
Description
