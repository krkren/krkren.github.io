Window.checkSignature

Function/Meaning
Start signature check
Type
Window class method
Syntax
checkSignature(filename, publickey, info)
Arguments
filename File to be checked
publickey Public key
info User information
Return Value
handler (Identification handler)
Description
The onCheckSignatureProgress event occurs during processing, and the onCheckSignatureDone event occurs upon completion.
For exe or dll files, the signature is embedded within the file. For files such as xp3, an external file with .sig appended to the original filename is processed as the signature.
