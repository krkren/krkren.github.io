BinaryStream.decompress

Function/Meaning
Decompress and copy data from another storage
Type
BinaryStream class method
Syntax
decompress(storage, elm)
Arguments
storage Source storage (an exception is thrown if it is the same as the currently open storage)
elm Dictionary for operation specification (in/out). Content is the same as copy().
Return Value
Number of bytes written
Description
Writes to the current stream position while decompressing the target storage using zlib/inflate.
An exception occurs if decompression fails.
The filter function is applied to the binary after decompression.
hash[out] is calculated for the binary after decompression/filtering.
