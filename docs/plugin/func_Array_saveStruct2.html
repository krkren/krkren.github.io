Array.saveStruct2

Function/Meaning
Save data in a format compliant with saveStruct
Type
Array class method
Syntax
saveStruct2(filename, utf8, newline)
Arguments
filename Destination file
utf8 Specifies output encoding. If true, UTF-8; if false, the current code page.
newline Line break code. 0:CRLF 1:LF
Return Value
None (void)
Description
