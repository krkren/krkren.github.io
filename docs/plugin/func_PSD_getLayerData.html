PSD.getLayerData

Function/Meaning
Reading layer data
Type
PSD class method
Syntax
getLayerData(layer, no)
Arguments
layer Destination layer
no Layer number
Can only be read if layer_type is layer_type_normal
In addition to the data content, the following properties are automatically set to the layer:
left Left coordinate
top Top coordinate
width Width
height Height
type Blend mode
opacity Opacity
visible Visibility status
imageLeft Becomes 0
imageTop Becomes 0
imageWidth Becomes width
imageHeight Becomes height
name name is set
Return Value
None (void)
