Window.setOverlayBitmap

Function/Meaning
Displays or clears the overlay bitmap
Type
Method of the Window class
Syntax
setOverlayBitmap(layer)
Arguments
layer The layer to copy to the bitmap image (clears the overlay if void or omitted)
Return Value
None (void)
Description
Covers the client area with a child window (a Windows window, not a Kirikiri one) using the specified image.
- An exception will occur if extended events are not enabled.
- While the overlay is displayed, no mouse messages are sent to the Kirikiri layer side.
- Effectively intended for use only when System.eventDisabled=true.
- Automatically cleared by TVP_WM_DETACH.
