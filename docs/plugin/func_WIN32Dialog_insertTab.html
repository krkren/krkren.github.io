WIN32Dialog.insertTab

Function/Meaning
Use tab control.
Type
WIN32Dialog class method
Syntax
insertTab(tab_id, pos, label)
Arguments
Return Value
None (void)
Description
Usage procedure is under consideration
insertTab(IDC_TAB, 0, "Basic"); // Add a tab
// deleteTab(IDC_TAB, 0); // Delete a tab
// deleteAllTab(IDC_TAB); // Delete all tabs
var dlg = new WIN32Dialog(); dlg.loadResource("resource.dll", "INNER"); // Dialog set to Border=None, Style=Child
selectTab(IDC_TAB, dlg); // Display
insertTab(IDC_TAB, 1, "Advanced");
var dlg2 = new WIN32Dialog(); dlg2.loadResource("resource.dll", "INNER2");
dlg.show(SW_HIDE);
selectTab(IDC_TAB, dlg2); // Switch
// Tab switching by the user occurs when notify->code == TCN_SELCHANGE and notify->idFrom == IDC_TAB in onNotify
// Get the tab index using getCurSel(tab_id), then set the corresponding dialog to SW_SHOW and others to SW_HIDE
// You must manage the relationship between tab indices and corresponding dialogs yourself
* Since the open for dlg and dlg2 is performed when selectTab is called, check if they are open using isValid when performing show(SW_HIDE).
