Scripts.equalStructNumericLoose

Function/Meaning
Compares objects. For dictionaries/arrays, the elements are compared recursively. Numbers are compared loosely (0.0 and 0 are treated as equal).
Type
Method of the Scripts class
Syntax
equalStructNumericLoose(obj1, obj2)
Arguments
obj1 Object 1
obj2 Object 2
Return Value
true if equal
Description
