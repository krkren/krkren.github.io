Clipboard.setAsBitmap

Function/Meaning
Sets the content of the layer to the clipboard as a bitmap
Type
Method of the Clipboard class
Syntax
setAsBitmap(layer)
Arguments
layer Layer
Return Value
None (void)
Description
When transferring to applications other than Kirikiri, the alpha value is cleared to 255. (It is recommended to composite the layer beforehand to remove the alpha channel).
When transferring to Kirikiri, the alpha value is preserved as is.
