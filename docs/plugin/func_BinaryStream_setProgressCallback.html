BinaryStream.setProgressCallback

Function/Meaning
Sets the progress callback
Type
Method of BinaryStream class
Syntax
setProgressCallback(callback)
Arguments
callback Callback function: function(storage, read_size) { return true_if_cancel; }
If void, the callback is cleared.
Return Value
None (void)
Description
Sets a function to be called back during copy/compress/decompress operations.
In the current implementation, the callback is triggered every 1MiB read and when the process reaches the end.
Returning true from the callback allows the operation to be canceled.
*The callback interval is subject to change.

By design, TJS processing blocks (freezes) during these operations, but by using plugins like windowExProgress, it becomes possible to accept cancellation requests from the user.
