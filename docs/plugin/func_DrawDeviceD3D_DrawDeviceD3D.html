DrawDeviceD3D.DrawDeviceD3D

Function/Meaning
Constructor
Type
Method of DrawDeviceD3D class
Syntax
DrawDeviceD3D(width, height)
Arguments
width Drawing area width
width Drawing area height
Return Value
None (void)
Description
The width and height correspond to the width/height of the primaryLayer in the conventional mechanism.
All created primary layers will be stretched and displayed to fit the area specified here.
Example:
drawDevice = new DrawDeviceD3D(800,400);
base1 = new Layer(this, null); base1.setSize(800,400); // Same resolution as screen
base2 = new Layer(this, null); base1.setSize(400,200); // Half the resolution of the screen
base3 = new Layer(this, null); base1.setSize(800,200); // Half resolution only vertically
