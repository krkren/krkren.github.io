Unzip.list

Function/Meaning
Retrieves information about files contained in the ZIP archive
Type
Method of the Unzip class
Syntax
list()
Arguments
Return Value
File list (array of file information)\nFile information is in the form of a dictionary and includes the following information:\n
\n
filename
Path name within the archive\n
uncompressed_size
Uncompressed size\n
compressed_size
Compressed size\n
crypted
Whether it is encrypted\n
deflated
Whether it is compressed\n
deflateLevel
Compression level\n
crc
CRC of the file\n
