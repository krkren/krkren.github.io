HttpRequest.sendStorage

Function/Meaning
Sends a request. Transmission and reception are executed asynchronously.
Type
Method of the HttpRequest class
Syntax
sendStorage(storage, storeStorage)
Arguments
storage File data to be sent
storeStrorage Destination file to save the response. If specified, the response property will not be set.
Return Value
None (void)
Description
No exception is thrown even in the event of an error, so please evaluate the result using readyState and status.
