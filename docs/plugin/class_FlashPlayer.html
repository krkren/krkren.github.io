FlashPlayer

FlashPlayer Class
An object that controls the playback of FlashPlayer on a layer.

ExternalInterface Functionality


　

◇Calling TJS methods from ActionScript



Any method of the object can be called via ExternalInterface from ActionScript within the playing SWF.

-----------------------------------------------------------
import flash.external.ExternalInterface;
trace(ExternalInterface.call("tjsFunction", 10, "test"));
-----------------------------------------------------------

Please note that even if an exception occurs on the Kirikiri side and the call fails, an exception will not be raised on the ActionScript side.
Exception information can be retrieved using getLastTJSError().

◇Calling ActionScript methods from TJS



With callFunction(), you can call ActionScript methods that have been exposed via ExternalInterface.

Preparing the function in ActionScript
-----------------------------------------------------------
import flash.external.ExternalInterface;
// Method to be called
function flashFunction(a:Number, b:String):String {
return StringUtil.substitute("{0}:{1}", a, b);
}
// Registration
ExternalInterface.addCallback("flashFunction", flashFunction);
-----------------------------------------------------------

Calling from TJS
-----------------------------------------------------------
var flash = new FlashPlayer(100,100);
flash.initMovie("sample.swf");
Debug.message(flash.callFunction("flashFunction", 1, "test"));
-----------------------------------------------------------

An exception will occur if the call fails.
