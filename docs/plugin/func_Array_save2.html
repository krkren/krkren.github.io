Array.save2

Function/Meaning
Saves data in a format conforming to the save method.
Type
Method of the Array class
Syntax
save2(filename, utf8, newline)
Arguments
filename Storage destination file
utf8 Output encoding specification. If true, UTF-8; if false, the current code page.
newline Newline code 0:CRLF 1:LF
Return Value
None (void)
Description
