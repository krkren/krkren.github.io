Layer.drawImageAffine

Function/Meaning
Affine transformation copy of an image
Type
Method of the Layer class
Syntax
drawImageAffine(src, sleft, stop, swidth, sheight, affine, A, B, C, D, E, F)
Arguments
src Source image (Image/Layer/filename)
sleft Left edge of the source rectangle
stop Top edge of the source rectangle
swidth Width of the source rectangle
sheight Height of the source rectangle
affine Type of affine parameters (true: transformation matrix, false: coordinate specification),
Return Value
Update region information (RectF)
Description
