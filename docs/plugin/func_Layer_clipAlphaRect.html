Layer.clipAlphaRect

Function/Meaning
Multiplies the layer's Alpha CHANNEL by the src Alpha CHANNEL
Type
Method of Layer class
Syntax
clipAlphaRect(dleft, dtop, src, sleft, stop, swidth, sheight, clear)
Arguments
dleft Left edge position
dtop Top edge position
src Source image containing alpha for multiplication
sleft Source left edge position
stop Source top edge position
swidth Source width
sheight Source height
clear Fills the area outside the operation range with this value (opacity)
If omitted or if the value is outside the 0-255 range, the area will not be filled.
Return Value
None (void)
Description
