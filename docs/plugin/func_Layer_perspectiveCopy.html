Layer.perspectiveCopy

Function/Meaning
Perspective transform copy
Type
Layer class method
Syntax
perspectiveCopy(src, left, top, width, height, x1, y1, x2, y2, x3, y3, x4, y4)
Arguments
src Source layer
left Source left
top Source top
width Source width
height Source height
x1 Top-left destination X
y1 Top-left destination Y
x2 Top-right destination X
y2 Top-right destination Y
x3 Bottom-left destination X
y3 Bottom-left destination Y
x4 Bottom-right destination X
y4 Bottom-right destination Y
Return Value
None (void)
Description
