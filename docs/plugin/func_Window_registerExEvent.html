Window.registerExEvent

Function/Meaning
Enable extended events
Type
Window class method
Syntax
registerExEvent()
Arguments
Return value
None (void)
Description
For this window instance, callbacks described as "■ Extended Event:" in the description will be enabled.

However, please note that the following callbacks will not occur if they do not exist in the instance at the time this function is executed.
(If they are missing, they will not be called even if the function is added to the instance later.)
onMoving Moving
onMove Moved
onResizing Resizing
onNcMouseMove Mouse movement in the non-client area
