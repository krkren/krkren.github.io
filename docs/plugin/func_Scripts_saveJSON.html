Scripts.saveJSON

Function/Meaning
Save data in JSON format
Type
Method of the Scripts class
Syntax
saveJSON(jsonFile, obj, utf8, newline)
Arguments
jsonFile Destination file
obj Object to be saved
utf8 Output encoding specification. If true, UTF-8; if false, the current code page.
newline Newline code 0:CRLF 1:LF
Return Value
None (void)
Description
