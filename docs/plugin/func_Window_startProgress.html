Window.startProgress

Function/Meaning
Starts progress processing. Display continues normally even while Kirikiri is in an execution block.
Type
Method of the Window class
Syntax
startProgress(init)
Arguments
init Initialization data (dictionary)
Return Value
None (void)
Description
Progress bar initialization parameters
progressBarEnable: Whether to display the progress bar (default: true)
progressBarStyle: Style of the progress bar (default: none)
progressBarLeft: Progress bar display position X (default: auto-centering)
progressBarTop: Progress bar display position Y (default: auto-centering)
progressBarWidth: Progress bar width (default: 1/3 of the window width)
progressBarHeight: Progress bar height (default: 1/10 of the window height)
progressBarColor: Progress bar color 0xRRGGBB (default: default color)
progressBarBackColor: Progress bar background color 0xRRGGBB (default: default color)

Cancel button initialization parameters
cancelButtonEnable: Whether to display the cancel button (default: true)
cancelButtonCaption: Text to display on the cancel button (default: Cancel)
cancelButtonLeft: Cancel button display position X (default: auto-centering)
cancelButtonTop: Cancel button display position Y (default: position at 3 times the height from the bottom)
cancelButtonWidth: Cancel button width (default: number of text characters * 12 + 8)
cancelButtonHeight: Cancel button height (default: 20)

Background image initialization parameters (unimplemented)
backGround: Background specification. Layer: refers directly to the layer's image. Numeric value: fills with the corresponding color. Default: none

Message initialization parameters (unimplemented)
messages: Message information or an array of such information
left: Display coordinate X
top: Display coordinate Y
text: Initial display text
size: Display size
color: Display color
shadowColor: Shadow color
shadowDistanceX: Shadow offset specification
shadowDistanceY: Shadow offset specification
font: Font specification
fontStyle: Font style
