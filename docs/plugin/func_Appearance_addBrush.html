Appearance.addBrush

Function/Meaning
Adds a brush
Type
Method of the Appearance class
Syntax
addBrush(colorOrBrush, ox, oy)
Arguments
colorOrBrush ARGB color specification or brush information (dictionary)
ox Display offset X
oy Display offset Y
Return Value
None (void)
Description
Brush information definition
Basic: Point specification is an array [x,y] or a dictionary %[x:x, y:y]
Rectangle specification is an array [x,y,width,height] or a dictionary %[x:x, y:y, width:w, height:h]
Color specification is an ARGB 32-bit integer value
Please refer to the GDI+ documentation for parameter details.

type: Specified by brush type BrushType

For BrushTypeSolidColor: *Same as specifying the color directly as ARGB.
color: Color specification (white if omitted)

For BrushTypeHatchFill:
hatchStyle: Hatch type (HatchStyleHorizontal if omitted)
foreColor: Foreground color (white if omitted)
backColor: Background color (black if omitted)

For BrushTypeTextureFill:
image: Image specification
wrapMode: Wrap pattern specification (WrapModeTile if omitted)
dstRect: Texture area rectangle specification (entire image if omitted)

For BrushTypePathGradient:
points: Array of point specifications
centerColor: Center color
centerPoint: Center point
focusScale: Focus scales specification [xScale, yScale] or %[xScale:, yScale:]
surroundColors: Surrounding color specification. Array of colors.
---- The following are common with BrushTypeLinearGradient
blend: Blend factor specification. Array of numbers [blendFactors, blendPositions] or a dictionary.
blendBellShape: Blend shape specification (bell shape) [focus, scale]
blendTriangularShape: Blend shape specification (triangular shape) [focus, scale]
gammaCorrection: Whether to enable gamma correction (true/false)
interpolationColors: [presetColors (color array), blendPositions (numeric array)]

For BrushTypeLinearGradient:
Common:
color1: Start color specification
color2: End color specification
wrapMode: Wrap pattern specification (WrapModeTile if omitted)
Point specification:
point1: Starting point
point2: Ending point
*The angle seems to be determined automatically.
Rectangle specification:
rect: Point specification. Top-left is start, bottom-right is end.
angle: Angle specification (valid only for rect specification)
isAngleScalable (Whether the angle specification is scalable)
mode: Mode specification (LinearGradientModeHorizontal if omitted). Valid when angle is not specified.
