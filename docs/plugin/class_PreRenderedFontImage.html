PreRenderedFontImage

Font image (*This class does not actually exist!)

Defines additional information to be added to the layer object returned by the callback.
Assign values to a standard layer object and pass it.
Only the alpha information in the (0,0)-(blackbox_x-1,blackbox_y-1) area of the layer's image is referenced, and it is resampled into 65 levels (0 to 64).

It is fine to reuse the same instance multiple times as the return value of the callback.
