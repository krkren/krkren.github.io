Window.onMoveSizeBegin

Function/Meaning
■Extended Event: Resize/Move Start Notification/End
Type
Method of the Window class
Syntax
onMoveSizeBegin()
Arguments
Return Value
None (void)
Description
Called when dragging of the frame or title bar starts, or when S or M is selected via Alt+Space (start notification), and when moving or resizing ends (end notification).
If you want to refer to the original window position or size during onMoving or onResizing, it is recommended to save them here.
Whether it is a resize or move cannot be determined due to the nature of WM_ENTERSIZEMOVE/WM_EXITSIZEMOVE.
