Layer.copyAlphaToProvince

Function/Meaning
Copies the layer's Alpha CHANNEL to the Province Image
Type
Layer class method
Syntax
copyAlphaToProvince(threshold)
Arguments
threshold Threshold value. If omitted or void, it copies as is.
If a numeric value is specified, it fills with 1 if the alpha is greater than or equal to the threshold, and 0 otherwise.
Return Value
None (void)
Description
