Layer.saveImage

Function/Meaning
Save image
Type
Method of Layer class
Syntax
saveImage(filename, mime, param)
Arguments
filename File name
mime Image format to save (image/jpeg, image/png, image/gif, etc.)
param Encoding parameters (dictionary)
compression, scanmethod, version, render, transform, quality (0-100), depth (1, 4, 8, 24, 32)
Return Value
None (void)
Description
