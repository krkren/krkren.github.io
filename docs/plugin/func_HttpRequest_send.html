HttpRequest.send

Function/Meaning
Sends a request. The sending process is executed asynchronously.
Type
Method of the HttpRequest class
Syntax
send(data, storeStorage)
Arguments
data Data to send
In case of octet: sent as is
In case of string: processed with the default encoding and sent
Others: no data is sent
storeStrorage File to save the response. If specified, the response property will not be set.
Return Value
None (void)
Description
No exceptions are thrown even in case of an error, so please judge based on readyState and status.
