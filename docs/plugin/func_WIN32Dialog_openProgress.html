WIN32Dialog.openProgress

Function/Meaning
Displays a progress dialog
Type
Method of WIN32Dialog class
Syntax
openProgress(id, window, dsapp, breathe)
Arguments
id Dialog item ID of the progress bar (the target updated by the progressValue property)
window [optional] Owner window (void = no owner, null = Application)
dsapp [optional] Whether to disable the application window while the progress is displayed
breathe [optional] Whether to "breathe" (process system messages) in the progressValue setter
Return Value
true if successful
Description
Do not specify WS_VISIBLE in the progress dialog style.
Always use closeProgress() when closing.
Note that for progress displays, no callbacks other than onInit will be triggered.
