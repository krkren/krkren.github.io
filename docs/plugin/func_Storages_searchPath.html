Storages.searchPath

Function/Meaning
Search for a path
Type
Storages class method
Syntax
searchPath(filename, searchpath)
Arguments
filename File name to search for
searchpath Search target paths (separated by ";" in local notation (e.g., c:\...); if omitted, the system default search path is used)
Return Value
void if not found; the full path of the file (file://./...) if found
Description
