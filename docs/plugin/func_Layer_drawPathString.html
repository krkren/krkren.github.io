Layer.drawPathString

Function/Meaning
Path-based drawing of strings
Type
Layer class method
Syntax
drawPathString(font, app, x, y, text)
Arguments
font Specifies the font using GdiPlus.Font
app Specifies the drawing style using GdiPlus.Appearance
x Drawing position X
y Drawing position Y
text Text to draw
Return Value
Update region information (RectF)
Description
