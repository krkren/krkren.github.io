Clipboard.setMultipleData

Function/Meaning
Registers multiple data formats to the clipboard at once
Type
Clipboard class method
Syntax
setMultipleData(data)
Arguments
data Specify the data in dictionary format.
Return Value
None (void)
Description
Data is specified as a dictionary containing multiple pairs of "data format: data body".
The supported data types are as follows:
- text: Text data
- tjs: TJS expression
- bitmap: Layer
