System.getDoubleClickTime

Function/Meaning
Get the system's double-click timeout period
Type
Method of the System class
Syntax
getDoubleClickTime()
Arguments
Return Value
Time (ms). Default is 500ms.
Description
Refer to the GetDoubleClickTime() API manual for details.
The allowable movement range for double-clicking can be obtained via CXDOUBLECLK/CYDOUBLECLK in getSystemMetrics.
Used when performing pseudo-double-click detection in software without using Layer.onDoubleClick (using a hardcoded time is usually fine, but this is provided just in case...).
