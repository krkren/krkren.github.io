Layer.copyExposure

Function/Meaning
Create image from summation buffer
Type
Method of Layer class
Syntax
copyExposure(stat)
Arguments
stat : Information dictionary for thresholding / If void, implicitly executes statExposure and uses that value.
The layer itself is overwritten as an image where the range from min to max for each element is mapped to 0 to 255.
If min >= max for an element, it is always returned as 0xFF.
Return Value
None (void)
Description
