Window.Window

Function/Meaning
Constructor
Type
Method of Window class
Syntax
Window(window, left, top, width, height)
Arguments
window Parent window
left X position
top Y position
width Width
height Height

At the time of creation, the Irrlicht device does not exist.
It will be initialized before the first event processing after creation,
enabling the driver, scene manager, and GUI to be handled.
Processing must wait for the onAttach() event.
Return Value
None (void)
