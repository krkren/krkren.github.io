SimpleHTTPServer.onRequest

Function/Meaning
Request processing (Event)
Type
SimpleHTTPServer class method
Syntax
onRequest()
Arguments
request = %[
method: Method type (GET/POST/...)
path: Request path
request: Raw request (%xx undecoded, includes everything after ?)
header: %[ Request headers (browser information, etc.) ]
form: %[ Form data ]
];
Return Value
response = %[\nstatus: Status code (optional, automatically 200 if no error)\ncontent_type: Content-Type name\nerror_type: Error type (effective when text/file/binary are absent)\nerror_desc: Error description (effective when text/file/binary are absent)\nredirect: URI for redirection (optional)\n\nThe following are exclusive (choose one, prioritized from top to bottom)\ntext: Text to return (automatically encoded with codepage)\nfile: File to return\nbinary: Binary to return (Array or Octet)\n];
Description
