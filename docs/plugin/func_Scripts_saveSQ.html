Scripts.saveSQ

Function/Meaning
Saves data in Squirrel format. This file starts with "return", so it can be read with dofile().
Type
Scripts class method
Syntax
saveSQ(filename, obj, utf8, newline)
Arguments
filename Destination file
obj Object to be saved
utf8 Output encoding specification. If true, UTF-8; if false, the current code page.
newline Line break code 0:CRLF 1:LF
Return Value
None (void)
