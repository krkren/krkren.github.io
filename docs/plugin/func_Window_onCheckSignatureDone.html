Window.onCheckSignatureDone

Function/Meaning
Event: Signature check completed
Type
Method of the Window class
Syntax
onCheckSignatureDone(handler, info, result, errormsg)
Arguments
handler Identification handler
info User information
result Result -2: Error, -1: Cancel, 0: Check failed, 1: Check successful
errormsg Error message if the result is an error
Return Value
None (void)
Description
