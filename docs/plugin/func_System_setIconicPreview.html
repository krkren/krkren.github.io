System.setIconicPreview

Function/Meaning
Force the taskbar and Alt+TAB thumbnail display to be the application icon on Vista and later
Type
System class method
Syntax
setIconicPreview(iconic)
Arguments
iconic If true, display the icon instead of the thumbnail / if false, clear the setting
Return Value
Whether it succeeded (always fails on XP and earlier)
Description
* DWMWA_FORCE_ICONIC_REPRESENTATION is set for the application window.
