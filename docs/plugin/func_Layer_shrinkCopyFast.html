Layer.shrinkCopyFast

Function/Meaning
Limited reduction copy (Shrinking by an integer factor & processing while ignoring alpha component)
Type
Method of Layer class
Syntax
shrinkCopyFast(src, shrink_x, shrink_y)
Arguments
shrink_x Horizontal reduction factor (1 or more)
shrink_y Vertical reduction factor (1 or more). If omitted or 0, the value of shrink_x is used.
Return Value
None (void)
Description
Transparent parts are ignored and always become opaque.
At the right and bottom edges, if the dimensions are not perfectly divisible, it may appear stretched by the fractional remainder of the 1-dot area.

* Note that the layer's own size is forcibly changed.
⇒ Equivalent to setImageSize(Math.ceil(src.imageWidth / shrink_x), Math.ceil(src.imageHeight / shrink_y))
