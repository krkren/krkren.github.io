SimpleDevice.updateToLayer

Function/Meaning
Draws the current state to a layer. It is scaled and copied from the back buffer to the drawing area of the specified layer. *Note: The layer's clipping region is ignored.
Type
Method of the SimpleDevice class
Syntax
updateToLayer(layer, sleft, stop, swidth, sheight)
Arguments
layer Destination layer
sleft X-coordinate of the top-left corner of the source area. *If the following are omitted, the entire area is used.
stop Y-coordinate of the top-left corner of the source area.
swidth Width of the source area
sheight Height of the source area
Return Value
None (void)
Description
