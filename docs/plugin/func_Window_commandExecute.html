Window.commandExecute

Function/Meaning
Execution of a command-line program with console output capture
Type
Method of the Window class
Syntax
commandExecute(target, param)
Arguments
target Program
param Arguments
Return Value
Process handle (-1 if failed). This can be passed to terminateProcess.
Description
The onCommandLineOutput event occurs for each line of console output.
The onShellExecuted event occurs when the process is finished.
