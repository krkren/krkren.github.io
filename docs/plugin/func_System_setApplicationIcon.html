```xml setApplicationIcon - Changes the application icon displayed in the taskbar, ALT+TAB, etc.

System.setApplicationIcon

Function/Meaning
Changes the application icon displayed in the taskbar, ALT+TAB, etc.
Type
Method of the System class
Syntax
setApplicationIcon(file)
Arguments
file Icon file or void (resets to the initial state if void or "")
Return Value
None (void)
Description
* Specifying files inside an archive is not allowed. If an exe or DLL is specified, the first icon will be used.
```
