Scripts.setEvalErrorLog

Function/Meaning
Suppress error log output of Scripts.eval
Type
Scripts class method
Syntax
setEvalErrorLog(enabled)
Arguments
enabled Enable output
Return Value
Original value
Description
If enabled is set to false, the evaluation process of Scripts.eval will be performed via TVPExecuteExpression, and logs will not be output when an exception occurs.
Note that specifications may change with Kirikiri version updates.
