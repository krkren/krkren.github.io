Process.commandExecute

Function/Meaning
Execution of a command-line program with console output capture
Type
Process class method
Syntax
commandExecute(target, param, folder)
Arguments
target Program
param Arguments
folder Execution folder (current folder if not specified)
Return Value
true if startup is successful
Description
The onOutput event occurs for each line of console output.
The onExecuted event occurs when the process is finished.
