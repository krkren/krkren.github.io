WaveSoundBuffer.getSample

Function/Meaning
Get sample value (old method)
Type
Method of the WaveSoundBuffer class
Syntax
getSample(n)
Arguments
n Number of samples to retrieve. If omitted, defaults to 100.
Return Value
Average value\n* Note: useVisBuffer=true must be set in advance.
Description
Retrieves the specified number of samples from the current playback position and returns their average value.
Samples with negative values are ignored.
