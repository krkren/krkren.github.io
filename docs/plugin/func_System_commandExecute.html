System.commandExecute

Function/Meaning
Executes a command line program and captures console output
Type
System class method
Syntax
commandExecute(target, param, timeout)
Arguments
target Program
param Arguments
timeout Timeout duration (ms)
Return Value
%[\nstatus: One of the strings: "ok", "error", or "timeout", \nstdout: Array of console output strings (separated by line breaks)\nexitcode: Exit code (only when status == "ok"),\nmessage: Error message (only when status != "ok")\n];
Description
No background processing is performed. Kirikiri will remain suspended until the target program finishes execution.
