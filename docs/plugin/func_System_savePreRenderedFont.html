System.savePreRenderedFont

Function/Meaning
Save pre-rendered font data to a file
Type
Method of the System class
Syntax
savePreRenderedFont(storage, characters, callback)
Arguments
storage The filename to save to
characters An array containing the characters (character codes) to be saved
callback Callback to retrieve information and images.
Must be a function that takes a character code as an argument and returns a layer (PreRenderedFontImage).
function(ch) { return layer; }
Return Value
None (void)
Description
