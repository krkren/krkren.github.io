System.getSystemMetrics

Function/Meaning
Get system metrics information
Type
Method of the System class
Syntax
getSystemMetrics(index)
Arguments
index The string representing the "*" part of SM_* (case-insensitive)
Return Value
Metrics value (content varies depending on index)
Description
Refer to the GetSystemMetrics() API documentation for details.
Since ARW_* is not defined for SW_ARRANGE, compare by specifying numeric values directly.
