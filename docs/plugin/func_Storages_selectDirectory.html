Storages.selectDirectory

Function/Meaning
Opens the folder selection dialog
Type
Method of the Storages class
Syntax
selectDirectory(params, title, window, rootDir)
Arguments
params Set parameters similar to selectFile
Return Value
Returns true if a folder is selected and the OK button is pressed, false if the Cancel button is pressed.\nparams.name: Specifies the folder name. If the OK button is pressed, the selected folder will be set to this member.\nparams.title: Displays the title of the dialog box.\nparams.window: Specifies the Window object that opens the dialog (if void, mainWindow is used; if unspecified, the desktop becomes the owner window. If the desktop is the owner, it is modeless).\nparams.rootDir: Specifies the root for folder selection (only folders under this directory will be displayed).
