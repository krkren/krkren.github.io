Window.storeHWND

Function/Meaning
Specifies whether to store HWND information.
Type
Property of the Window class
Description
When a value is specified for this, a file named "executable_filename.key_value" is created, and the current HWND information will be output inside it.
If you want to send messages from an external application, you can refer to the HWND recorded in this file.
The content is automatically updated when the HWND changes, such as during full-screen mode.
Because of this, external applications must re-reference the HWND value every time they send a message.
