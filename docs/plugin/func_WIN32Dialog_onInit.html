WIN32Dialog.onInit

Function/Meaning
Callback for WM_INITDIALOG
Type
WIN32Dialog class method
Syntax
onInit(msg, wparam, lparam)
Arguments
msg DlgProc message == WM_INITDIALOG
wparam DlgProc WPARAM
lparam DlgProc LPARAM
Return Value
None (void)
Description
Please note that the callback is called on the owner.
If owner.onInit is undefined, nothing happens.
