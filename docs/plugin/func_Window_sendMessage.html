Window.sendMessage

Function/Meaning
Sends a message.
Type
Window class method
Syntax
sendMessage(key, message)
Arguments
key Identification key (string: note that it is limited to 256 characters)
message Message (string)
Return Value
None (void)
Description
Sends a WM_COPYDATA message to all running Kirikiri instances.
The meaning of the identification key and message should be defined appropriately for each application.
