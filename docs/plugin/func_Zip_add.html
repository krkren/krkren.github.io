Zip.add

Function/Meaning
Adds a file to the archive
Type
Zip class method
Syntax
add(srcfile, destfile, commpressLevel, password)
Arguments
srcfile File to add (Kirikiri storage name)
destfile Name in the archive (Path name within the archive)
deflateLevel Compression level 0 (no compression) to 9 (maximum compression). Defaults to 6 if omitted.
password Encryption password specification
Return Value
true if addition is successful
Description
