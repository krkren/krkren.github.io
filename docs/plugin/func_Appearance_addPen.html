Appearance.addPen

Function/Meaning
Add Pen
Type
Method of the Appearance class
Syntax
addPen(colorOrBrush, widthOrOption, ox, oy)
Arguments
colorOrBrush ARGB color specification or brush information (dictionary)
widthOrOption Pen width or pen information (dictionary)
ox Display offset X
oy Display offset Y
Return Value
None (void)
Description
Pen information definition: If widthOrOption is a dictionary, it becomes a detailed information definition.
Please research the parameters by referring to the GDI+ documentation.

width: Pen width specification
alignment: Alignment: PenAlignmentCenter when omitted
compoundArray: Compound array specification. Numerical array
dashCap: Specification of the dash cap style. DashCapFlat when omitted
dashOffset: Specification of the dash offset. 0 when omitted
dashStyle: Dash style. If an array, it's user-defined (numerical array). Default is DashStyleSolid
startCap: Specification of the cap style at the start position. LineCapFlat when omitted, AdjustableArrowCap when a dictionary
endCap: Specification of the cap style at the end position. LineCapFlat when omitted, AdjustableArrowCap when a dictionary
lineJoin: Join shape specification. LineJoinMiter when omitted
miterLimit: Miter limit specification (numeric)

AdjustableArrowCap: For those who are unsatisfied with LineCapArrowAnchor
width (arrow width), height (arrow height), and filled (triangle if true, trident/three-pronged if false) can be specified.
