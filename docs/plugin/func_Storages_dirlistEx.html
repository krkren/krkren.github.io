Storages.dirlistEx

Function/Meaning
Retrieves a list of files and detailed information for the specified directory
Type
Method of the Storages class
Syntax
dirlistEx(dir)
Arguments
dir Directory name
Return Value
An array containing a list of file information\n[ %[ name:File name, size, attrib, mtime, atime, ctime ], ... ]\nNote that unlike dirlist, a trailing "/" is not added to the name for folders (use 'attrib' to distinguish).
