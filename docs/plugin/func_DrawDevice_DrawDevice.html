DrawDevice.DrawDevice

Function/Meaning
Constructor
Type
Method of DrawDevice class
Syntax
DrawDevice(width, height)
Arguments
width Back buffer width
width Back buffer height
The height and width correspond to the width/height of the primaryLayer in the conventional mechanism.
All created primary layers will be stretched and displayed relative to the area specified here.
Example:
drawDevice = new Irrlicht.DrawDevice(800,400);
base1 = new Layer(this, null); base1.setSize(800,400); // Same resolution as screen
base2 = new Layer(this, null); base1.setSize(400,200); // Half the resolution of screen
base3 = new Layer(this, null); base1.setSize(800,200); // Half resolution in height only

The Irrlicht device does not exist at the time of creation.
It is initialized when drawing processing as DrawDevice begins (at the time of SetTargetWindow call), enabling the driver, scene manager, and GUI.
Processing must wait for the onAttach() event before proceeding.
Returns
None (void)
