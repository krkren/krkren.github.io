ActiveX.ActiveX

Function/Meaning
Constructor
Type
ActiveX class method
Syntax
ActiveX(name, win, left, top, width, height)
Arguments
progIdOrCLSID Specify the Program ID or CLSID as a string.
name Specify the Program ID or CLSID as a string. *Note that the CLSID format is different from the WIN32OLE specification; it follows the CAxWindow::CreateControl format.
win If specified, it is created on top of that window. If omitted, it becomes an independent window.
left Display coordinates. If a window is specified and this is omitted, it defaults to the top-left of the window's client area.
top Display coordinates. If a window is specified and this is omitted, it defaults to the top-left of the window's client area.
width Display size. If a window is specified and this is omitted, it defaults to the size of the parent window's client area.
height Display size. If a window is specified and this is omitted, it defaults to the size of the parent window's client area.
Return Value
None (void)
Description
