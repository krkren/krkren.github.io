BinaryStream.setFilter

Function/Meaning
Sets a filter DLL
Type
Method of the BinaryStream class
Syntax
setFilter(dll)
Arguments
dll Target DLL file (unsets if void)
Return Value
None (void)
Description
Sets a filter using an external DLL for copy/compress/decompress data.
When a function name is specified in elm.filter, the function exported from this DLL in the following format will be used as the filter:
void (__stdcall *FilterProc)(uint32 hash, uint64 offset, void *buffer, uint32 length);
