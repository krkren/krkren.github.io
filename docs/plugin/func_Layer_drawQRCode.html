Layer.drawQRCode

Function/Meaning
Adjusts the layer size and writes a QR code according to the specified value (string or integer)
Type
Layer class method
Syntax
drawQRCode(value, ecLevel, qrVersion, autoExtent, maskPattern)
Arguments
value The value to be converted into a QR code (string or integer)
ecLevel Error correction level (default: 0 / 0 (min) to 3 (max))
qrVersion Capacity size group (default: 0 / 0 to 2)
autoExtent Whether to auto-expand (default: true (automatically expands to an appropriate size if the capacity is too small for the value))
maskPattern Masking pattern (default: -1 (auto-selection))
Return Value
None (void)
Description
