Storages.copyFileNoNormalize

Function/Meaning
Copies the specified file in Kirikiri's storage space without path normalization (the destination file name is not converted to lowercase)
Type
Method of the Storages class
Syntax
copyFileNoNormalize(from, to, failIfExist)
Arguments
from Source file
to Destination file
failIfExist true if the operation should fail if the file exists, false to overwrite
Return Value
true if the file was successfully copied
Description
