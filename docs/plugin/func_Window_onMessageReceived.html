Window.onMessageReceived

Function/Meaning
Event: Message received.
Type
Window class method
Syntax
onMessageReceived(key, message)
Arguments
key Identification key (String: note that it is limited to 256 characters)
message Message (String)
Return Value
None (void)
Description
Called when a WM_COPYDATA message is received from another Kirikiri or an external application.
Since the caller is locked until this call completes, please ensure that the process finishes promptly.
