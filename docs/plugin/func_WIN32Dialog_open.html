WIN32Dialog.open

Function/Meaning
Opens the dialog
Type
WIN32Dialog class method
Syntax
open(window)
Arguments
window Parent window (Kirikiri Window class); if omitted or void, no parent is assigned.
Return Value
None (void)
Description
A dialog template must have been previously created using makeTemplate, or a dialog resource must have been loaded using loadResource.
Note that data from loadResource takes precedence over makeTemplate.
