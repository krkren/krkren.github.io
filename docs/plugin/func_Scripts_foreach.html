Scripts.foreach

Function/Meaning
Iteration through all members of an object
Type
Method of the Scripts class
Syntax
foreach(obj, func, args)
Arguments
obj Target object (dictionary or array)
func Callback function. It is called in the form func(key, value, args*).
If the context is null, such as in anonymous functions, it automatically operates in the "this" context.
The process is interrupted if the function returns any value.
args Additional arguments
In the case of an array, it is simply a reference via a loop. For dictionaries and objects, it results in a call to iTJSDispatch2::EnumMebers.
Return Value
The value returned upon interruption
