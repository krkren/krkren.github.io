Layer.getDiffRect

Function/Meaning
Gets the differential rectangle of the layer.
Type
Layer class method
Syntax
getDiffRect(base)
Arguments
base The base image to be used as the source for comparison.
Return Value
A dictionary in the format %[ x, y, w, h ], or void (if the images are completely identical).\n (An exception is thrown if "base" is not the same size as the instance itself.)
