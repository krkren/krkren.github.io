Layer.copyRaster

Function/Meaning
Raster copy processing
Type
Method of the Layer class
Syntax
copyRaster(src, maxh, lines, cycle, time)
Arguments
src Source layer. Processing will not be performed unless it is the same size as the target layer.
maxh Maximum amplitude (pixels)
line Number of lines per cycle
cycle Cycle specification (msec)
time Current time
Return Value
None (void)
Description
