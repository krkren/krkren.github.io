BinaryStream.readI8

Function/Meaning
Read {1, 2, 4, 8} bytes as a numeric value
Type
BinaryStream class method
Syntax
readI8()
Arguments
Return Value
The numeric value read / void if it could not be read due to the end of the stream, etc.
Description
*Values other than 8 bytes are read as unsigned (no sign extension is performed).
*void is returned only if not even 1 byte could be read.
Note that if at least 1 byte but fewer than the specified number of bytes are read, a value with the missing parts filled with 0 will be returned.
