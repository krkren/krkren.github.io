Scripts.compileStorageSQ

Function/Meaning
Compiles a Squirrel script from a file
Type
Method of the Scripts class
Syntax
compileStorageSQ(filename, store, endian)
Arguments
filename Source file to be compiled
store Destination file for the binary closure
endian Endianness specification 0: default 1: little 2: big
Return Value
Error string or void
Description
