Layer.drawImage

Function/Meaning
Draws an image. The copy destination position takes the source image's Bounds into account, and the size is specified in pixels.
Type
Layer class method
Syntax
drawImage(dleft, dtop, src)
Arguments
x Copy destination origin X
y Copy destination origin Y
image Copy source image (Image/Layer/Filename)
Return Value
Update region information (RectF)
