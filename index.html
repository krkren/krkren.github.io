Kirikiri Z

About Kirikiri Z

Kirikiri Z is the successor version of Kirikiri 2.
By using Kirikiri Z, you can create 2D games and applications.
It is used in many commercial ADV/novel games.
As an application, it is used in e-mote/Emofuri.
It can be used for various purposes beyond just games.

KAG3 is a system for developing ADV/novel games using Kirikiri Z.
krwidgets is a widget toolkit for creating GUI applications using Kirikiri Z.

Kirikiri 2 is located near the middle of this page.

Operating Environment
  • CPU : Pentium III 800MHz or higher (1GHz or higher recommended)
  • OS : Windows Vista, 7, 8, 10
  • Memory : 128MB or higher (1GB or higher recommended)
  • VIDEO : 65536 colors (TrueColor, 16bit) or higher (16,777,216 colors (FullColor, 32bit) recommended)
  • DirectX 9 or later required
  • Windows Media Player 9 or later required during WMV playback
  • Windows 7 or later required during H.264 playback

Operating requirements vary depending on the features used. This does not mean that all software created with Kirikiri Z will work practically in this environment.
Please do not copy and paste the above operating environment; the actual operating environment should be confirmed and set by the developer.

Note: The above content has changed significantly compared to Kirikiri 2, but the conditions are basically the same except for the OS. However, considering the current situation, the operating environment of Kirikiri 2 is considered too low, so it is raised here.
